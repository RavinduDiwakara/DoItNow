# DoitNow - Get Things Done

DoitNow is a clean and efficient Android task management application designed to help users organize their daily activities, track progress, and boost productivity.

## 🚀 Features

- **Task Management**: Easily add, view, and manage your daily tasks.
- **Progress Tracking**: Statistics for Total, Done, and Pending tasks to keep you motivated.
- **Priority Levels**: Categorize tasks with High, Medium, and Low priority levels.
- **User Authentication**: Secure Sign In and Sign Up functionality to protect your data.
- **Personal Profile**: Manage your user information and track your member status.
- **Modern UI**: Clean, intuitive interface built with Material Design components.

## 🛠️ Built With

- **Language**: Java
- **UI Framework**: Android XML (Material Components)
- **Database**: SQLite (via DatabaseHelper)
- **Architecture**: MVC
